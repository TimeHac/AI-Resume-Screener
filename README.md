## 📌 Overview

This project builds a resume screening system that:
- Accepts a resume as **plain text or PDF upload**
- Cleans and vectorizes the text using **TF-IDF (unigrams + bigrams)**
- Classifies it into one of **5 job categories** using a trained **SVM classifier**
- Returns the prediction with **confidence scores** for all categories via a REST API

**Categories:** Data Scientist · Web Developer · Software Engineer · HR · Marketing

## 🚀 Quick Start

### 1. Clone & Setup

```bash
git clone https://github.com/YOUR_USERNAME/ai-resume-screener.git
cd ai-resume-screener

python -m venv venv
source venv/bin/activate        # Windows: venv\Scripts\activate
pip install -r requirements.txt
```

### 2. Generate Data & Train Model

```bash
# Step 1: Generate the synthetic training dataset
cd data
python generate_dataset.py
cd ..

# Step 2: Train and save the model
python train_model.py
```

Expected output:
```
SVM  →  Test Accuracy: 88.40%
Logistic Regression  →  Test Accuracy: 87.20%

✓ Best model: SVM (88.40%)
  5-Fold CV: 87.96% ± 0.42%
✓ Model saved to model/resume_classifier.pkl
```

### 3. Run the Flask App

```bash
python app.py
```

Open **http://localhost:5000** in your browser.

## 🌐 API Reference

### `POST /predict`

Classify a resume. Accepts **JSON** or **file upload**.

**Option A – JSON body:**
```bash
curl -X POST http://localhost:5000/predict \
  -H "Content-Type: application/json" \
  -d '{"resume_text": "Python machine learning TensorFlow data analysis..."}'
```

**Option B – File upload (PDF or TXT):**
```bash
curl -X POST http://localhost:5000/predict \
  -F "file=@my_resume.pdf"
```

**Response:**
```json
{
  "category": "Data Scientist",
  "confidence": 92.4,
  "all_probabilities": {
    "Data Scientist": 92.4,
    "Software Engineer": 4.1,
    "Web Developer": 1.8,
    "Marketing": 1.1,
    "HR": 0.6
  },
  "word_count": 312,
  "status": "success"
}
```

### `GET /health`
```json
{
  "status": "ok",
  "model_loaded": true,
  "categories": ["Data Scientist", "HR", "Marketing", "Software Engineer", "Web Developer"]
}
```

## 🧠 Model Details

| Component | Details |
|---|---|
| Vectorizer | TF-IDF, unigrams + bigrams, top 15,000 features |
| Classifier | Linear SVM (`C=1.0`, `probability=True`) |
| Baseline | Logistic Regression (multinomial) |
| Dataset | 2,500 synthetic resumes (500 per class) |
| Train/Test Split | 80% / 20%, stratified |
| Cross-Validation | 5-Fold, ~88% mean accuracy |
